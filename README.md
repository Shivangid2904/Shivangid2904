### Projects

**[SafeHer AI](YOUR_SAFEHER_REPO_LINK_HERE)** *(In Progress)*
AI-powered women's safety prototype — risk scoring (0–100) using ML models, Dijkstra-based safe route optimization, and real-time location alerting.

**[Intelligent Question Answering System](https://github.com/Shivangid2904/Intelligent-Question-Answering-System)**
RAG pipeline using FAISS vector search and Sentence Transformers for semantic retrieval from PDFs. Runs a local LLM via Ollama — fully private, no API key required. Hybrid relevance scoring with session-persistent history via Streamlit.

**[CVE Vulnerability Intelligence Platform](https://github.com/Shivangid2904/cve-vulnerability-intelligence-platform)**
Serverless cybersecurity platform on AWS — automates CVE ingestion, classifies severity (Critical / Medium / Low), and sends real-time alerts via SNS. Built with Lambda, DynamoDB, Cognito, EventBridge, and CloudTrail.

**[Exoplanet Habitability Prediction](https://github.com/Shivangid2904/Exoplanet-Habitability-Prediction)**
Trained and evaluated machine learning models on NASA exoplanet datasets to predict planetary habitability.

**[Context-Aware Emotion AI](https://github.com/Shivangid2904/context-aware-emotion-ai)**
ML system that predicts emotional intensity from text and contextual signals, with uncertainty-aware recommendations for supportive actions.
